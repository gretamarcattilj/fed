Struttura base

Gli elemtenti base di una semplie pagina html sono:

  • DOCTYPE
  • apertura pagina tramite il comando < html >
  • creazione della sezione head
  • creazione della sezione body
    • < p > per i paragrafi
    • < br > per creare uno spazio

    Perchè html ?

    Html è utile perc reare pagine web o intefacce di applicazioni. Il programma può poi essere arricchito con scelte di stile tramite CSS ( un programma che vedremo in seguito )
